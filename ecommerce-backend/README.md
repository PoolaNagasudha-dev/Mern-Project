# E-commerce Backend API

This is the backend for an E-commerce website built using the MERN stack.

## Features

- User Registration
- Product CRUD APIs
- MongoDB database connection
- JWT Authentication (planned)
- Error handling middleware

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT
- bcrypt.js
